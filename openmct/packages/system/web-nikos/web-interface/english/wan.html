WAN configuration

Here you can setup your connection to the internet or to a different network.

Run DHCP client: This will allow the OpenMCT to ask for all the information it needs automatically. Your ISP or existing router should have a DHCP server running.

Run DSL client: If the WAN port is connected to a DSL or cable modem directly, you often need to specify data given to you by your provider. This option lets you run PPPoE and other types of DSL and cable connections.
Login and password: These are the username and password you have gotten from your ISP. (e.g. Hanz and 1234)
Dial on demand: A DSL connection will only be established if requested by a PC in the network. The connection will be terminated after it has not been used for Idle Time seconds
Idle time (in seconds): Use this option if you have a time-limited internet access. The DSL connection will be disconnected after no data traffic is detected within this amount of time (e.g. 600 seconds)

Use fixed IP address: This allows you to manually enter you network information.
IP address: This is your fixed IP address (e.g. 82.61.170.3)
Subnet mask: This is your subnet mask (e.g. 255.255.255.0)
Default gateway: This is your default gateway address, (e.g. 123.212.14.12)
DNS Servers: These are your DNS servers, seperated by a space (e.g. 123.212.14.12 123.212.14.13).
