User

New: Pressing the New button gives you a dialogue to add new users.

Properties: Selecting a user and pressing the Properties button will allow you to edit the user settings. (e.g. password)

Delete: Selecting a user and pressing the Delete button will delete the user. This will also delete the user folder by the same name, but not any other folders created were the user might have saved data.
