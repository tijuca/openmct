Mail Server

You can receive and send Mail directly from and to internet. To recieve mail you have to open the SMTP to the outside.
The received mail is stored in a mailbox for each user.
The mail can be read with any POP3 or IMAP client in the LAN. Just set the POP3/IMAP address in your mail software setup (f.e. MS Outlook). to the address of the OpenMCT. Username and password are the same as for OpenMCT login.

Run mail server: Switch the mail server on

Open IMAP/POP3/SMTP server: This will allow users to access the IMAP, POP3 and SMTP server from the outside through the WAN port. You can use Dynamic DNS (DDNS) or the IP address provided by your ISP to reach your OpenMCT. DDNS is recommended.

Fetchmail: This software polls all accounts you set up in a given intervall. The mails are stored in the mailbox of the local user you select. Port: These are the TCP ports where the servers listen for a connection.

Standard ports:
SMTP Port 25
IMAP Port 143
IMAPS Port 993
POP3 Port 110
POP3S Port 995
