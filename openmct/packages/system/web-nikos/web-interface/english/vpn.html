VPN connections

The server uses IPsec tunnels to create VPN connections. The listed connections are shown with Name, Status, Action, Remote Host and Remote Net.

Add: If you want to add a permanent new connection, please press Add Connection.

properties: If you want to see and edit the properties of an existing connection, select the connection and press Properties.

Delete: If you want to delete a connection, select the connection and press Delete Connection.

Show status: If you wish to see the status of all the connections, press Show status of all connections.
