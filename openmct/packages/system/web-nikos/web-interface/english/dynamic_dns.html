Dynamic Domain Name System


The Dynamic Domain Name (DDNS) System gives the ability to provide an official DNS name such as "www.yourname.com" to the (dynamic) IP address you

 have gotten from your ISP. You can then reach your OpenMCT over the internet at any time using www.yourname.com, even when the IP address given to you by your ISP changes. You will need to register at one of the listed providers. Please see their websites (same as their name) for further details.

Run dynamic DNS client: Check this if you want to enable the Dynamic DNS client

Service: Several different Service providers are supported. Register at the website of one of the service providers and fill in the required information.

Host: This is the domain name you registered at the DNS service provider

Username: This is the username you registered at the DDNS service provider

Password: This is the password you registered at the DDNS service provider

Retype password: This is the password you registered at the DDNS service provider again.
