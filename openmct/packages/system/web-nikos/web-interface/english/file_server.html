Services - File Server
  (CIFS/SMB)
  To make your data avaible through the WAN port to the outside
  using Windows file sharing (CIFS/SMB), turn this option on.
   
  This is a security risk, since users from the internet can also
  see your files.
   
  Make sure you have an additional firewall (your existing router
  for instance) blocking this traffic between you and the internet!
     
     
  Open Windows File Sharing (CIFS/SMB) on the WAN port
   
 
File Server (CIFS/SMB)

This will allow you to use Windows file sharing (CIFS/SMB) through the WAN port. If you allow this, please be aware anyone on the network you attach the WAN port of your OpenMCT to (ie the internet) will be able to use your OpenMCT via Windows file sharing. Make sure you only do this if you connect to an existing router and that router blocks Windows file sharing traffic from the internet. Most routers will block this type of traffic automatically.
