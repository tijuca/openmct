WLAN access list

Access list policy: You can choose how the hardware addresses in the list should be handled. Whether the list should be ignored (open to every client), whether the list members are the only ones to gain access to the OpenMCT (allow all in list) or whether the list members are the only ones to be rejected by the OpenMCT (deny all in list).

Access Control List: Here you can enter a list of wireless hardware addresses to which the access list policy applies.
