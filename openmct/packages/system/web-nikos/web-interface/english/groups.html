Groups

A group is a tool to quickly give access to a folder to multiple users. Add the users to a group and give the group access rights to one or more folders.

New: Pressing the New button gives you a dialogue to add a new group.

Properties: Selecting a group and pressing the Properties button will allow you to edit the group settings and add or remove users.

Delete: Selecting a group and pressing the Delete button will delete the group, but all users will be left intact. Access rights given to the group will be revoked! Users will not be able to access folders or files they are not authorized to via an other group or on a personal basis!
