Wireless LAN

enable WLAN: You can choose to enable or disable the WLAN access point.

Network Name: This is the name of the wireless network for the OpenMCT. This name is used to connect to the OpenMCT.

Encryption Mode: The wireless access point can be configured to use no encryption, WEP or WPA. To enable it without encryption is very dangerous and should be used for short tests only. WEP is okay in most cases, but does not satisfy high security standards. WPA is the best choice.

WEP hex key: Type in your WEP key here in hex format. It must consist of either 10 hex-digits (WEP 40bit) or 26 hex-digits (WEP 104bit). Do not use passphrases as is offered by some WLAN client programs!

WPA-PSK key: Type in the WPA private shared key here.

Encryption: Choose which WPA encryption modes are accepted by the acces point.
