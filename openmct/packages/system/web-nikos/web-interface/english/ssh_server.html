SSH Server

The SSH server is a secure terminal client. It can be used to do advanced Configuration using the linux command line. Please do not use it unless you need it. It is a security risk! All users with an account will be able to access the OpenMCT. If you do not know what it is, leave it off.

Run SSH server: Start/stop the server

Open SSH server to the outside: This will allow users to access the SSH server through the WAN port, ie from the internet.
