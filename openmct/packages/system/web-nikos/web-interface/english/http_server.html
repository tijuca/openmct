HTTP Server

These settings adjust the web server. The webserver cannot be switched off, since it is used to run the configuration. It is always accessible throught the LAN ports. The web server used is a version of Apache.

Open HTTP server: This allows the http server to be reached through your WAN port. IP adresses whose source address is located somewhere on the internet can now configure the machine and use the HTML interface to the files. (If enabled, it is recommended to use SSL encryption as well).

Use SSL encryption: This will allow SSL encrypted pages. (https) You can reach your OpenMCT from "https://Address.of.OpenMCT" instead of "http://Address.of.OpenMCT".
