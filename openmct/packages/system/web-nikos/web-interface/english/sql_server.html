SQL Server

This server is a MySQL server. You can create and use databases via SQL. This can be done with a MySQL client or with a PHP script.

Run SQL server: Switch the SQL server on.

Open SQL server: This will allow users to access the SQL server from the the outside through the WAN port, ie from the internet. You can use Dynamic DNS (DDNS) or the IP address provided by your ISP to reach your OpenMCT. DDNS is recommended. Note that the user needs appropriate SQL permissions, too.

TCP Port: This is the TCP port where the SQL server listens for connections.

Administration page: By clicking on the link you get to a configuration system where databases and users can by configured.

Administrator password: If something went wrong, you can set the administrator account anew. After that you can enter the administration page again.
