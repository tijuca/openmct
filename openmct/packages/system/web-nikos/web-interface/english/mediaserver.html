Media Server

This server is a media server based on the Universal Plug and Play Protocol (UPnP).

Enable Media Server: Switch on the media server.

open media server for the outside: If enabled the media server can be accessed via the WAN port.

Content folders: User folders which are searched for content, default is all user folders. Hold CTRL-C to select multiple folders.

Enable Internet Radio: Enables Shoutcast Internet Radio

Enable picture rescaling: Enables the server to rescale pictures. This needs a lot of power and time.

Enable audio format resampling: Enables resampling of audio formats. This options needs a lot of power and time.

Language: Select the userinterface langugage of the advanced server pages.

Scan for content: Start a new scan in content folders.

Advanced: Switch to advanced configuration pages.
