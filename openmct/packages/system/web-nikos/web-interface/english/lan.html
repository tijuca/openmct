LAN configuration

Here you can setup your local network or LAN ports.

IP address: This is the IP address your OpenMCT uses in your LAN (e.g.192.168.1.1)

IP subnet mask: This is your subnet mask (e.g. 255.255.255.0)

Workgroup: This is the name of the Windows workgroup in which your OpenMCT is located. (e.g. MSHOMENET)

Router name: The name of the OpenMCT in the LAN. This name can be used to access the web configuration (e.g. http://nas) instead of using the IP address. This is also the name your OpenMCT appears as under windows file sharing. Be aware! Changing this name can lead to problems with drive mappings!
